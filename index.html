The Prisoners Dilemma - Simulating the Cycle Strategy

You can use this webpage to simulate the success rate of the Prisoners Experiment using the cycle strategy given different parameters. You may adjust the number of Prisoners, Boxes, and Trials run and it will return the success rate with the given inputs.

WARNING: "Run Full Scale Trial" will run 1000 trials with 1000 prisoners, for each box 0-1000. Results will be exported to a text file. This will take an EXTREMELY long time to run.
